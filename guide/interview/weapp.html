# 小程序

# 【1】小程序登录流程

  1. wx.login获取临时登录凭证code
  2. 发送code给后端，后端通过code，appid，appsecret调用微信接口，返回openid和session-key；
  3. 后端通过openid和session-key生成token返回给前端
  4. 前端把后端返回的token缓存起来

# 【2】静默登录和非静默登录

  • 静默登录：wx.login
  • 非静默登录：wx.getuserProfile
